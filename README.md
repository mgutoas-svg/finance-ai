# Finance AI

Sistema financeiro pessoal com foco em acompanhamento de movimentações, criação de metas e apoio à tomada de decisão.

## Objetivo do sistema

Este projeto tem como objetivo ajudar no controle financeiro pessoal, permitindo:

- Registro de entradas e saídas financeiras
- Visualização do histórico financeiro
- Criação de metas financeiras com valor e prazo
- Acompanhamento do progresso das metas
- Geração de alertas e insights para apoiar decisões financeiras

## Evolução planejada

O desenvolvimento deste sistema será feito de forma gradual, priorizando simplicidade e clareza.

Etapas previstas:

1. Estrutura inicial do backend e armazenamento de dados
2. Registro e consulta de movimentações financeiras
3. Criação e acompanhamento de metas financeiras
4. Geração de alertas baseados no comportamento financeiro
5. Inclusão de análises mais avançadas e apoio inteligente à tomada de decisão
6. Preparação para uso como aplicação web (SaaS)
