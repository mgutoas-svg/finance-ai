## Objetivo do sistema

Este projeto tem como objetivo ajudar no controle financeiro pessoal, permitindo:

- Registro de entradas e saídas financeiras
- Visualização do histórico financeiro
- Criação de metas financeiras com valor e prazo
- Acompanhamento do progresso das metas
- Geração de alertas e insights para apoiar decisões financeiras
# Finance AI

Sistema financeiro pessoal com foco em acompanhamento de movimentações, criação de metas e apoio à tomada de decisão.
